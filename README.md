## ⬇️ Installation

### Windows

1. Go to the [**Releases**](../../releases) page
2. Download the latest Ormycut.exe file
3. Double-click to run — no installation required

> ⚠️ If Windows SmartScreen warns you, click **"More info" → "Run anyway"**
> This happens because the app is not signed with a paid certificate.

---

### Requirements

- Windows 10 / 11
- No Python needed — everything is bundled
